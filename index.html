JavaScript-"The Bare Necessities"

Put your skills to the test with this multiple choice quiz on the fundamentals of JavaScript!

Description

  • Press the start button to begin the quiz.
  • Once the START button has been pressed the timer will begin counting down.
  • Click the checkboxes to select your answer.
  • Correct Answers will move the user on to the next question and the timer will continue as usual.
  • Incorrect answers will result in 5 seconds being subtracted from the timer.
  • The time remaining at the end of the quiz will be the user's score.
  • Once the game is over, add your Initials to be placed on the Highscores list!
